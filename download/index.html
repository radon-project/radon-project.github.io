Installation methods

Choose how to install

📦 PyPI (Recommended)

Install from the Python Package Index using pip.

pip install radon-lang

🔧 From Source

Clone the repository and run directly with Python.

git clone https://github.com/radon-project/radon.git
cd radon
python radon.py

📥 Release Archive

Download a specific version as a zip or tarball from GitHub.

# Download from GitHub Releases
# Extract and run:
python radon.py
